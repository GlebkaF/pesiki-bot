# Ralph

Автономный AI-агент для разработки. Работает в Docker sandbox с Claude Opus 4.5.

## Использование

```bash
# Одна итерация (интерактивно)
./ralph.sh

# AFK режим - 10 итераций
./ralph.sh 10

# Print mode (для логов)
./ralph.sh 5 -p
```

## Как это работает

1. Ralph читает `PRD.md` и `progress.txt`
2. Находит следующую незавершённую задачу `[ ]`
3. Реализует её
4. Отмечает `[x]` в PRD
5. Обновляет progress.txt
6. Делает git commit
7. Повторяет (если указано несколько итераций)

## Файлы

| Файл | Описание |
|------|----------|
| `ralph.sh` | Скрипт запуска (всегда в sandbox) |
| `PRD.md` | Документ с задачами |
| `progress.txt` | Лог выполненных задач |

## Cursor Commands

Альтернатива CLI, работает прямо в IDE:

- `/ralph` - одна итерация
- `/ralph-plan` - создать/редактировать PRD
- `/ralph-status` - проверить прогресс

## Модель

По умолчанию используется `opus-4.5`. Можно изменить в `ralph.sh`:

```bash
MODEL="opus-4.5-thinking"  # С расширенным reasoning
MODEL="sonnet-4.5"         # Быстрее, дешевле
```

## Docker Sandbox

Все команды выполняются в изолированном контейнере:
- Агент может устанавливать пакеты, запускать скрипты
- Твоя система защищена от случайных повреждений
- Рабочая директория автоматически монтируется
