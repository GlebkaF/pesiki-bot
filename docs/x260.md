# Сервер x260

## Подключение

- SSH хост: `192.168.0.118`, юзер `gleb`

## Деплой

- Автоматический: push в `main` → GitHub Actions → `docker compose up --force-recreate` на x260
- Секреты (токены, ключи, прокси) хранятся в GitHub Repository Secrets и пробрасываются через workflow

## Дебаг

- Бот работает в Docker-контейнере `pesiki-bot`
- Логи: `docker logs pesiki-bot`
- Переменные окружения: `docker exec pesiki-bot printenv`
