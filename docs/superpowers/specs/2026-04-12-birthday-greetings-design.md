# Birthday Greetings — Design

## Цель

Бот автоматически поздравляет игроков с днём рождения в групповом чате. Поздравление душевное, короткое, с нативно вплетённой дота-статистикой за прошедший год жизни. Генерируется через OpenAI с учётом `botAttitude` игрока.

## Данные

Поле `birthday?: string` (формат `"YYYY-MM-DD"`) добавляется в интерфейс `Player` в `config.ts`.

Известные даты рождения:

| dotaName | birthday |
|---|---|
| zladey | 1993-11-10 |
| Marinad | 1993-02-06 |
| Shootema | 1997-11-03 |
| Curiosity | 1998-04-12 |
| Aoba | 1997-01-20 |

У остальных игроков поле отсутствует — они не участвуют в поздравлениях.

## OpenDota данные за год

Для именинника запрашиваем три эндпоинта с параметром `date=365`:

- **`/players/{id}/wl`** — wins/losses за год
- **`/players/{id}/heroes?sort=games`** — топ героев за год (берём топ-3)
- **`/players/{id}/totals`** — средние kills, deaths, assists, GPM

Из этого формируется саммари:
- Всего матчей за год, винрейт %
- Топ-3 героя (имя, кол-во игр, винрейт на герое)
- Средние KDA, GPM
- Возраст (вычисляется из `birthday`)

## AI-генерация поздравления

Промпт для OpenAI:
- Роль: бот дота-компании, поздравляет игрока с ДР
- Контекст: имя, возраст, `botAttitude`, дота-статистика за год
- Инструкции: душевное, короткое (3-5 предложений), дота-данные вплетены естественно, без списков/буллетов, не слащаво
- Формат вывода: plain text

Структура сообщения в чат:
```
🎂 <a href="tg://user?id={telegramId}">Имя</a>

{AI-сгенерированное поздравление}
```

## Архитектура

### Новый файл: `src/birthday.ts`

Функция `checkAndSendBirthdayGreetings(bot)`:
1. Получает сегодняшнюю дату (MM-DD)
2. Фильтрует `PLAYERS` — у кого `birthday` совпадает по MM-DD
3. Для каждого именинника:
   - Запрашивает OpenDota статистику за 365 дней (wl, heroes, totals — параллельно)
   - Резолвит имена героев
   - Формирует промпт с данными + `botAttitude`
   - Вызывает OpenAI
   - Отправляет сообщение в чат с mention

### Изменения в `src/index.ts`

Новый cron: `0 19 * * *` (19:00 MSK) — вызывает `checkAndSendBirthdayGreetings(bot)`.

### Изменения в `src/config.ts`

- Поле `birthday?: string` в интерфейсе `Player`
- Даты рождения в массиве `PLAYERS`

### Изменения в `src/opendota.ts`

Новые функции:
- `fetchWinLoss(playerId, days)` — GET `/players/{id}/wl?date={days}`
- `fetchTopHeroes(playerId, days)` — GET `/players/{id}/heroes?date={days}&sort=games`

Существующие `fetchPlayerTotals` уже поддерживает фильтр по дате.

## Обработка ошибок

- OpenDota недоступен → поздравление без статистики (fallback-промпт без дота-данных)
- OpenAI недоступен → лог ошибки, сообщение не отправляется
- Нет именинников сегодня → ничего не происходит
